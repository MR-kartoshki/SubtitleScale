# Changelog

All notable changes to SubtitleScale are documented in this file.

## [1.1.0] - 2026-02-26

### Added
- X/Y subtitle offset controls.
- Profile presets: `Default`, `Cinematic`, `Accessibility Large`.
- Quick scale preset cycling with `F3 +` and `F3 -`.

### Changed
- Universal release jar naming and metadata for Minecraft `1.21.9` through `1.21.11`.
